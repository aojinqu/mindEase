# mindEase
## 1.介绍
MindEase 是一款面向大学生的 AI 辅助情绪日记与情绪支持 Android 应用。项目目标是在 Android Studio + Java 技术栈下，完成一个低门槛、低压力、可持续使用的情绪记录与支持系统，覆盖“记录 - 分析 - 建议 - 社区”主流程。

## 2. 项目背景与动机

大学生常常面临学业压力、截止日期、社交关系、职业规划和个人生活等多方面压力。很多学生希望记录自己的情绪，或者寻找一个低压力的方式表达想法，但现有心理健康类应用往往存在以下问题：

- 过于医疗化或专业化
- 操作复杂，日常使用门槛高
- 更适合成年人或泛用户，而非学生群体
- 提供泛泛建议，缺乏可执行性

MindEase 的目标是：

- 提供简单快速的每日情绪记录方式
- 通过图表和日历帮助用户理解情绪变化
- 提供低压力、匿名的表达空间
- 基于 AI 或规则分析提供短小、实用、非医疗化的支持建议

本产品定位为“日常情绪陪伴工具”，而不是医学诊断工具。

---

## 3. 产品目标

### 3.1 主要目标

- 帮助用户以低门槛方式记录每日情绪
- 帮助用户通过可视化图表理解情绪趋势
- 提供轻量级 AI 情绪分析与自助建议
- 提供匿名、安全的表达空间
- 提高用户对自身情绪状态的觉察能力与管理习惯

### 3.2 非目标

- 不提供医疗诊断
- 不替代专业心理咨询或治疗
- 不在第一版中做成完整社交平台
- 不在第一版中支持 iOS
